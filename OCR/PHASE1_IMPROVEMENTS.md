## ?? How to Use New Features

### **Quick Save Workflow**

1. Complete OCR extraction
2. Click **?? Save** button (or press `Ctrl+S`)
3. Choose location and filename
4. Click Save
5. ? Status bar shows confirmation

### **Resume Previous Session**

1. Click **File** ? **Open...** (or press `Ctrl+O`)
2. Select your saved `.txt` file
3. Continue adding more text or edit existing

### **Clear and Start Fresh**

1. Click **??? Clear** button (or press `Ctrl+L`)
2. Confirm in dialog
3. Output cleared, ready for new OCR

### **Monitor Progress**

- Watch the **progress bar** fill during batch processing
- See **page count** update in status bar
- Read **status messages** for current operation
